  Innovative User Interface  
     
 

ELT Pro offers many innovative interfaces for logistics providers to be more productive and efficient in today’s competitive global market. ELT Pro is committed to provide the best solutions that you need in order to be successful in the shipping industry. Click here to learn more about the ELT Pro’s innovative features.
* Standard form interface makes it simple and easy to work Since users will spend most of their time working to create forms like airway bills, ELT Pro screens are specially designed for users to feel as if they’re looking at these forms. It doesn’t require hours of training even if you are using it for the first time. You can relate and apply many things you already knew in ELT Pro.  
* Easy screen layout allows user-friendly navigationELT Pro makes logistics process more user friendly. Air, ocean, export, import, accounting, report, search and admin integrate seamlessly with one another. Group in such a way that user can navigate from one task to the next with a click of a mouse.  
* Favorite task for frequently used taskYou can create shortcut to most frequently used task by drag and drop in to a folder. You can then work in this folder without having to move around screens. 
* Open multiple pages in one browser instead of open multiple browsers Within the same browser, you can open new airway bill, search previous airway bill, look up customer info, check rate and add port info all at the same time. This feature is analogous to opening and working in several Microsoft Excel worksheets at the same time.  
* Recent work saves most recent work without having to go back This feature will automatically save the most recent work that you did. If you create a booking, customer profile, airway bill and invoice, you can see everything by going into the recent work folder. This feature is automatically available to you so you don’t have to do anything. 
* Hyperlinks to other related pages In many cases you have to look up airway bill, invoice and customer info from your current screen. In ELT Pro, you can refer to other related pages without having to go to that screen. The hyperlinks from current page allows you to view other related pages for your convenience. 
* Collaborative messaging board makes excellent communication toolELT Pro allows you to share important message about new trade regulation, change of customer’s contact info and last minute cancellation of airline with your colleagues by using message board. When you login to ELT Pro, this will be the first thing that you will see on your screen so you can’t miss the important announcement.  
* Acrobat PDF and Microsoft Excel file optionELT Pro supports Acrobat PDF and Microsoft Excel. PDF and Excel files are convenient to work with because they are universally adapted. This is another benefits of using ELT Pro that you can simply view, save, print and email your work in PDF and Excel format.
